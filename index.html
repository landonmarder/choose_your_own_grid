Flexible Grid

Download

Idea: Use the grid to position elements that you want in the same row. To do so, make a div with a class 'row.' Then, inside the div, have child divs that have classes with the size of the column. For example, 'column-1-of-2' is a class that makes the div 1/2 of the original div 'row' container.
